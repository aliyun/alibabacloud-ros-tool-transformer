# Usage

Use these pages as focused references for each `rostran` command group:

- [Transform templates](./usage/transform.md): Convert Terraform, CloudFormation, Excel, ROS, and ROS Terraform templates.
- [Format templates](./usage/format.md): Format ROS templates consistently.
- [Manage transform rules](./usage/rules.md): View, update, list, and reset transform rules.
- [Start the web service](./usage/web-service.md): Run the browser-based playground and configure Terraform project caching.

For the complete command list, run:

```bash
rostran --help
```
