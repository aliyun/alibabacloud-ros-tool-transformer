# ROS Template Transformer

[ROS](https://help.aliyun.com/product/28850.html) Template Transformer（阿里云ROS模板转换工具）用于转换和格式化模板。

- 将 **Excel、Terraform 或 AWS CloudFormation** 转换为 **ROS** 模板。
- 将 **ROS** 模板转换为 **Terraform** 模板。
- 将 **ROS** 模板格式化。

浏览器版 [Playground](/playground) 可直接体验 CloudFormation 转 ROS 和 ROS
格式化。Terraform、Excel、ROS 转 Terraform 等完整流程请通过 `rostran server start`
启动本地 Web 服务体验。

该项目使用 Python 语言开发，需要安装 Python 3.9 或以上版本。
