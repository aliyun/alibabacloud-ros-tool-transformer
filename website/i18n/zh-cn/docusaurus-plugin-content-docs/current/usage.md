# 使用

按命令组查看 `rostran` 的用法：

- [转换模板](./usage/transform.md)：转换 Terraform、CloudFormation、Excel、ROS 和 ROS Terraform 模板。
- [格式化模板](./usage/format.md)：统一格式化 ROS 模板。
- [管理转换规则](./usage/rules.md)：查看、更新、列出和重置转换规则。
- [启动 Web 服务](./usage/web-service.md)：运行浏览器版 playground，并配置 Terraform 项目缓存。

查看完整命令列表：

```bash
rostran --help
```
